Windows 2008-R2 Server - RAID

This hands-on activity walks you through Windows 2008-R2 Server - RAID. It focuses on practical server configuration tasks you can replicate in a virtual lab environment. Follow each step demonstrated in the video, apply the settings on your own machine, and document your results.

Step-by-Step Guide

  1. Launch your Windows Server 2008-R2 virtual machine or lab setup.
  2. Follow each configuration step shown in the video carefully.
  3. Pause and replicate each action on your own server instance.
  4. Verify the service or feature is working correctly after configuration.
  5. Take screenshots and document your process for activity submission.
