About

This project is a learning portal designed for students and IT beginners studying Windows operating systems and server administration.

Each video has its own dedicated page with an embedded player, a detailed description, a step-by-step learning guide, and a direct link to the original YouTube source.

The site is built as a dynamic single-page application and published via GitHub Pages for easy sharing.
